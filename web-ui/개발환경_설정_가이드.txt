========================================================
  PLC Monitor Web UI - 개발환경 설정 가이드
========================================================

[작성일] 2026-03-28
[프로젝트 경로] Claude_UI\web-ui


========================================================
  1. Node.js 설치 (최초 1회만)
========================================================

1) https://nodejs.org 접속
2) LTS (안정 버전) 다운로드 후 설치
   - 설치 옵션은 전부 기본값 그대로 Next 클릭
3) 설치 확인 (명령 프롬프트 또는 PowerShell):

   node -v
   npm -v

   버전 번호가 나오면 정상


========================================================
  2. 프로젝트 파일 복사
========================================================

USB, Git, 클라우드 등으로 Claude_UI 폴더 전체를
새 PC에 복사합니다.

* 단, web-ui\node_modules 폴더는 복사하지 않아도 됩니다.
  (용량이 크고, 3단계에서 자동으로 다시 생성됩니다)


========================================================
  3. 패키지 설치
========================================================

명령 프롬프트(cmd) 또는 PowerShell에서:

1) web-ui 폴더로 이동
   cd C:\Users\[사용자이름]\source\repos\Claude_UI\web-ui

2) 패키지 설치 (인터넷 필요)
   npm install

   * 이 명령어가 package.json을 읽고
     필요한 라이브러리를 전부 자동 설치합니다.
   * node_modules 폴더가 생기면 완료


========================================================
  4. 개발 서버 실행
========================================================

같은 web-ui 폴더에서:

   npm run dev

실행하면 아래와 같은 메시지가 나옵니다:

   VITE ready in XXX ms
   -> Local:   http://localhost:5173/

브라우저에서 http://localhost:5173 을 열면
PLC 모니터링 대시보드가 표시됩니다.

* 서버 종료: 명령 프롬프트에서 Ctrl + C


========================================================
  5. 코드 수정 방법
========================================================

web-ui\src 폴더 안의 파일을 수정하면 됩니다.

[주요 파일]
- src\App.jsx    : 대시보드 화면 구조 (React 컴포넌트)
- src\App.css    : 대시보드 스타일 (색상, 레이아웃)
- src\index.css  : 전체 공통 스타일 (다크 테마 등)

* 개발 서버(npm run dev)가 켜져 있으면
  파일 저장 즉시 브라우저에 반영됩니다. (HMR 기능)
  새로고침할 필요 없음.


========================================================
  6. 문제 해결
========================================================

[문제] npm 명령어가 안 된다
-> Node.js가 설치되지 않은 상태. 1단계부터 진행.

[문제] npm install 중 에러 발생
-> node_modules 폴더 삭제 후 다시 npm install

[문제] 포트 5173이 이미 사용 중
-> npm run dev -- --port 3000
   (다른 포트 번호로 실행)

[문제] 브라우저에 아무것도 안 뜬다
-> 주소가 http://localhost:5173 맞는지 확인
-> 명령 프롬프트에서 에러 메시지 확인


========================================================
  요약 (새 PC에서 할 일)
========================================================

  1. Node.js 설치    (nodejs.org)
  2. 프로젝트 복사    (Claude_UI 폴더)
  3. npm install     (web-ui 폴더에서)
  4. npm run dev     (개발 서버 시작)
  5. 브라우저 열기    (localhost:5173)
